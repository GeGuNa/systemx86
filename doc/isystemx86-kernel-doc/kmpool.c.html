kmpool.c

Ihsoh, 2015-05-14




文件描述

内核内存池。
这个内存池是通过libs/mempoollib实现的。
使用kmpool_malloc或kmpool_calloc分配的内存块必须使用kmpool_free释放内存。

Public
kmpool_calloc

函数描述

使用内核内存池分配内存块。
并且可以指定需要分配内存块的数量。
这些内存块是连续的。
分配的总尺寸为count * size。

参数描述

count, uint32, IN

需要分配内存块的数量。

size, uint32, IN

需要分配的内存块的尺寸。

返回值描述

void *

成功则返回指向第一个内存块的指针，否则返回NULL。
