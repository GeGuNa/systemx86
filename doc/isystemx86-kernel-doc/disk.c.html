_ahcisym2portno

函数描述

AHCI设备符号转换为端口号。

参数描述

symbol, CASCTEXT, IN

AHCI设备符号（SA、SB、SC、...、SZ）。 盘符缓冲区必须大于或等于DISK_SYMBOL_BUFFER_SIZE。

返回值描述

uint32

如果AHCI符号不合法则返回0xffffffff，否则： SA => 0 SB => 1 SC => 2 ... SZ => 25
