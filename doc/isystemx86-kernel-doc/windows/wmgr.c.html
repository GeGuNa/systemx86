_pad

函数描述

如果字符串的长度小于_ITEM_MAX_LEN，
则用空格填充字符串，使其长度等于_ITEM_MAX_LEN。
如果字符串的长度大于_ITEM_MAX_LEN，
则截断字符串，使其长度等于_ITEM_MAX_LEN，并且使最后三个字符变为“...”。

参数描述

text, ASCTEXT, IN OUT

指向要填充的字符串的缓冲区的指针。
